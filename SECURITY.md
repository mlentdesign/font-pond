# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Font Pond, please report it responsibly rather than opening a public issue.

Contact the author directly via [LinkedIn](https://www.linkedin.com/in/melanielent) with details of the vulnerability. Please allow reasonable time for a response before any public disclosure.

## Scope

This policy applies to the Font Pond site at https://mlentdesign.github.io/font-pond/. As a static site with no user accounts or backend, the attack surface is limited — but responsible reports are still welcome.
