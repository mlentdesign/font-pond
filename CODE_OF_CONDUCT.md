# Code of Conduct

This repository is a solo portfolio project and is not open to public contribution. Any interaction with this repository — including issue comments or public-facing communication — is expected to remain respectful and professional.

## Standards

- Be respectful in any issue comments or public interactions
- Do not use this space to solicit collaboration, request features aggressively, or pressure the author
- Do not attempt to circumvent the license terms stated in [LICENSE](./LICENSE)

## Enforcement

The author reserves the right to close, lock, or remove any interaction that violates these standards without notice.
